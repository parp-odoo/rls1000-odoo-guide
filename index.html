Requirements

Before starting, make sure you have the following assets ready:

  • An RLS1000 weighing scale
  • A Windows computer connected to the same network
  • The RLS Software installed on the computer
  • RLS Software download: Download RLS Software

Connect the Scale to Wi-Fi

Watch Configuration Video Follow our step-by-step video guide for Wi-Fi setup.

Access the Scale Configuration Menu

  1. Press and hold the [SET] key for approximately 3 seconds.
  2. The scale enters the system configuration mode.
  3. Use [↑] and [↓] keys (or the numeric keypad) to navigate.
  4. Press [PRN/ENTER] to confirm.

Configure Wi-Fi Settings

  1. Open the SET WIFI menu.
  2. Select SSID and enter your Wi-Fi name.
  3. Open Wi-Fi Security Mode.
  4. Select WPA2-PSK/AES (or your router's security mode).
  5. Enter the Wi-Fi password.

Configure Network Settings

Windows network settings reference: Microsoft Documentation

  1. Open the SET IP menu.
  2. Configure the scale IP address.
  3. Configure the subnet mask.
  4. Configure the gateway address.
Important: The scale's IP address must be within the same subnet as your router and must not conflict with other devices on the network.

Restart and Verify Connectivity

  1. Save the settings and restart the scale.
  2. Wait until the Wi-Fi icon appears on the screen.

    3. Successful Wi-Fi connection indicator on the scale display.

  3. Open the RLS Software on your PC.
  4. Enter the scale name and assigned IP address.
  5. Click Check Connection.

Locate the "Check Connection" button in the RLS Software.

Confirmation of a successful connection (Transfer Status OK).

Configure a Custom Barcode Format

The scale must generate barcodes that align with Odoo's barcode nomenclature for weight-based products.

  1. Open the RLS Software.
  2. Click the Settings (gear) icon.
  3. Navigate to the Custom Barcode tab.
  4. Select an unused barcode slot (e.g., 103).
  5. Enter the following format: 21IIIIIWWWWWW
  6. Set Right Digit to 3.
  7. Save the configuration.
  8. Click Download to push the settings to the scale.

Setting up the barcode format in RLS Software.

Configure Products in Odoo

Ensure each product in Odoo has a barcode prefix that matches your scale's configuration (e.g., beginning with 21).

For detailed instructions, refer to the Odoo Barcode Documentation.

Export Product Data from Odoo

  1. Navigate to the Products menu in Odoo.
  2. Select the products you wish to sync.
  3. Click Action → Export.
  4. Select the following fields:
    • Product Name
    • Barcode
    • Sales Price
  5. Choose XLSX format and click Export.

Prepare the PLU Data File

Format your exported XLSX data according to the required PLU table below.

Download Sample PLU Format (XLSX)

Field Description
Product Name The name displayed on the scale's screen.
PLU No The unique lookup number for the product.
Price The unit price of the product.
Unit Weight unit (kg or g).
Barcode Type Must match your custom barcode slot (e.g., 103).

Verify the Integration

  1. Place a product on the scale.
  2. Enter its PLU number (e.g., 1, 3, or 13).
  3. Press [PRN/ENTER] to print the label.
  4. In Odoo Point of Sale, scan the printed barcode.
Success Check: The product should appear in the POS cart with the correct weight and calculated total price.
